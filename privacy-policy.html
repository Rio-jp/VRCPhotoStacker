Privacy policy for VRChat Photo Manager

1. データの保存について

1. About Data Storage

VRChat フォトマネージャーは、すべてのデータをお使いのパソコンのローカルストレージにのみ保存します。データが外部のサーバーに送信されることはありません。

VRChat Photo Manager stores all data only on your computer's local storage. Data is never sent to any external servers.

当アプリケーションは、以下の情報をローカルに保存します:

The application locally stores the following information:

  • VRChatで撮影した画像
  • 画像のメタデータ（撮影日時、ワールド名など）
  • アプリケーション設定
  • ユーザーが追加したタグやメモ
  • Images captured in VRChat
  • Image metadata (capture date, world name, etc.)
  • Application settings
  • Tags and notes added by the user

2. データセキュリティ

2. Data Security

当アプリケーションはインターネット接続を必要とせず、データの送受信を行いません。そのため、お客様のプライバシーは完全に保護されています。

This application does not require an internet connection and does not send or receive data. Therefore, your privacy is completely protected.

ソフトウェアのアップデート確認のみ、オプションでインターネット接続を使用することがあります。この機能はいつでも設定から無効にすることができます。

Only software update checks may optionally use an internet connection. This feature can be disabled at any time from the settings.

3. 第三者提供

3. Third-party Sharing

当アプリケーションはお客様のデータを第三者に提供することはありません。すべてのデータ処理はお客様のコンピューター上でローカルに行われます。

This application does not provide your data to any third parties. All data processing is done locally on your computer.

4. プライバシーポリシーの更新

4. Privacy Policy Updates

当プライバシーポリシーは、必要に応じて更新される場合があります。重要な変更がある場合は、アプリケーション内で通知いたします。

This privacy policy may be updated as necessary. In case of significant changes, we will notify you within the application.
